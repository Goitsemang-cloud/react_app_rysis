You need to install nodejs from this site : https://nodejs.org/en/

To check if your Node is installed, in your cmd type 'node -v'
To install project packages you will need to 'npm install' inside the root files directory (C:\reactapp)
To run the project you will need to run 'npm run start' inside the root file directory (C:\reactapp)
